# DebugAI — Code Debugging Assistant

A fast, free, browser-based AI assistant that debugs your code instantly. No backend, no server — just open the HTML file and start debugging.

## Features
- Detects syntax, logic, and runtime errors
- Explains bugs in simple language
- Provides fixed code instantly
- Supports Python, JavaScript, Java, C++, TypeScript, Go
- 4 debug modes: Full Analysis, Explain, Fix, Optimize
- File upload support
- Powered by Groq (free tier available)

## How to Use
1. Get a free API key at [console.groq.com](https://console.groq.com)
2. Open `debugai-groq.html` in your browser
3. Enter your API key once (saved automatically)
4. Paste your code and hit **Debug**

## Tech Stack
- Frontend: HTML, CSS, JavaScript (no frameworks)
- AI: Groq API (Llama 3.3 70B)
- No backend required

## Models Supported
- Llama 3.3 70B *(recommended)*
- Llama 3 8B *(fastest)*
- Mixtral 8x7B
- Gemma 2 9B
